# AI Vision Pro: Advanced Real-Time Face Analysis

A **powerful, browser-based** AI system for real-time face detection, landmarks, age/gender estimation, and expression analysis. Built with **Vanilla JavaScript**, **face-api.js**, and **CDN-only** dependencies.

## 🚀 Live Demo

Experience the system instantly by visiting the live deployment:

**[🚀 View Live Demo](https://yeasin4745.github.io/face-detection-system/)**

## ✨ Core Features

| Feature | Description | Implementation Detail |
| :--- | :--- | :--- |
| **Real-time Face Detection** | High-performance face tracking with `TinyFaceDetector`. | face-api.js |
| **68 Face Landmarks** | Visualizes 68 specific points on the face with precision. | face-api.js |
| **Age & Gender Estimation** | Real-time AI analysis of age and gender. | face-api.js |
| **Expression Analysis** | Detects emotions (happy, sad, angry, surprised, etc.). | face-api.js |
| **Multiple Face Tracking** | Tracks and analyzes multiple people simultaneously. | face-api.js |
| **Responsive Design** | Fully optimized for mobile, tablet, and desktop. | Tailwind CSS |
| **Interactive Tour** | Guided walkthrough for new users. | Driver.js |
| **Privacy First** | All processing happens locally in your browser. | Client-side only |

## 🛠️ Technology Stack

- **Vanilla JavaScript**: Core application logic.
- **face-api.js**: Neural network-based face analysis.
- **Tailwind CSS**: Utility-first styling for a professional look.
- **Driver.js**: Interactive user onboarding.
- **Lucide Icons**: Modern, lightweight vector icons.

## 📦 Deployment

This project is optimized for **Vercel**. To deploy:

1. Push this repository to your GitHub account.
2. Import the repository in [Vercel](https://vercel.com).
3. Click **Deploy**.

No build steps or environment variables are required.

## 📖 How to Use

1. Click **"Start Detection"** to enable your camera.
2. Grant camera permissions when prompted.
3. View real-time analytics on the right panel (or below on mobile).
4. Use the **Help Icon** in the header to restart the interactive tour.
